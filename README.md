# photo-designer-lib


Library for designing marketing images. 

It can work in browser or on server. 

**Note: For usage on server, install [node-canvas](https://github.com/Automattic/node-canvas)**

## Testing

### Browser

```bash
npm run serve # and open console in browser
```

### Server

```
node ./example.js
```
